Synthesis of COnsists as Rolling Energy micro-grids (v0.2)

A toolset for exploring and optimizing energy, power, cost and emissions for rail transportation. Select a below category to begin using the interface

Routes

Routes downloaded from publically available databases to use as examples for power and emission analyses. These routes include elevation, gradient, curvature, and maximum speed details.

Select

Consists

Define or select a train consist to perform power and emission analyses over routes. Each consist includes a list of freight cars and locomotives.

Select

Analysis and Trade Studies

Simulation tools to calculate energy, power, cost and emissions of consists over routes. The Analysis tab allows one to select a route and consist to perform a power and emission analysis. The Trade studies tab allows one to select multiple routes and consists to support data exploration.
